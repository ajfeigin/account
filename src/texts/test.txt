
Account is a tiny language for making really simple calculators inside of HTML documents without getting in the way. It should take about five minutes to learn if you know how to use (1) a spreadsheet; and (2) a content management system.

Every expression is of the form:

name=pairs

A name is of the form a-z_.

A pair is a name and a value, separated by a colon.

The first pair is the command pair and the rest, if any, are qualifying pairs.

A value can be:
- A Decimal number like -12.5
- A String like 'Abraham Lincoln\'s pancake breakfast'
- A List like [1,2,3]
- A Language like (10 * cats), i.e. some math.
- {{itswitch=switch:['on','off']}}

The first thing to understand is that there are ranges, which get shown as sliders.

{{number=range:[1,15], default:5}}

And then you can do math, like this is that squared times two: {{square=math:(2 * (number ^ 2))}} or {{cubed=math:(number ^ 3)}} cubed. Or raised to its own power, ala {{exp=math:(number^number)}}
